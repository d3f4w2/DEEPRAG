# RAGAS 评测说明（新链路）

当前评测已**完全切换为 RAGAS**，不再使用旧的“关键词命中 / 证据命中”规则链路。

## 核心指标

固定输出以下 5 项：

1. `忠诚度`（Faithfulness）
2. `上下文召回率`（Context Recall）
3. `答案准确度`（Answer Correctness）
4. `平均 token`
5. `平均延迟`

---

## 数据集格式

评测集 JSON 顶层字段：

- `版本`
- `说明`
- `问题列表`（数组）

每题最少包含：

- `id`
- `问题`
- `参考答案`（RAGAS 必需）

可选字段：

- `参考上下文`（string[]）
- `期望证据文件`（string[]）
- `来源类型`
- `query_style`
- `query_length`

---

## 一键运行评测

```bash
python 评测/运行小评测.py --run-name baseline_ragas
```

默认数据集是：

`评测/ragas评测集_40题.json`

---

## 构建评测集

### 1) 推荐：用 RAGAS 自动生成

```bash
python 评测/构建RAGAS评测集.py --mode ragas --size 40 --output 评测/ragas评测集_40题.json
```

说明：

- 会读取知识库 markdown 文档
- 通过 RAGAS testset generator 自动生成问答样本
- 自动转换成本项目评测格式

### 2) 兼容：从旧评测集转换

```bash
python 评测/构建RAGAS评测集.py --mode legacy-convert --legacy-dataset 评测/小评测集_40题.json --output 评测/ragas评测集_40题.json
```

说明：

- 仅用于迁移启动，不是高质量 gold answer
- 推荐后续仍用 `--mode ragas` 重建一份更难的数据集

---

## 常用参数

- `--base-url`：后端地址（默认 `http://127.0.0.1:8000`）
- `--provider`：覆盖 provider（问答与 RAGAS 评分都使用该 provider）
- `--max-questions`：只跑前 N 题
- `--timeout`：单题超时（秒）
- `--compare`：传入上一轮 `*_汇总.json` 生成对比表

---

- `*_明细.json`
